# RASPBIAN Distro + PiTFT HAT 2.2":
##########################################
Download pre-packaged RASPIAN BIN from:
  https://learn.adafruit.com/adafruit-2-2-pitft-hat-320-240-primary-display-for-raspberry-pi/easy-install
  http://adafruit-download.s3.amazonaws.com/2015-02-16-raspbian-pitft22_150312.zip
and burn it to the SD-card
  

  
  
##########################################  
# INSTALL PI-TFT:
##########################################
SHOULD NOT BE REQUIRED IF TFT-enabled BINARY IS USED!
RUN:
     sudo bash install_pitft22.sh
OR: Following instructions from:
    https://learn.adafruit.com/adafruit-2-2-pitft-hat-320-240-primary-display-for-raspberry-pi/easy-install

configure TFT:
   sudo adafruit-pitft-helper -t 22
